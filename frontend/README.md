This is the FireCrow FCv1 frontend, built with [Next.js](https://nextjs.org).

## Getting Started

For local development:

1. Start the backend on `http://localhost:8000`.
2. Use `frontend/.env.local`, or copy `frontend/.env.example` to `.env.local`.
3. Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The dashboard uses `NEXT_PUBLIC_API_URL`, which points locally to `http://localhost:8000/api/v1`.

The application shell, typography, and status system follow the FireCrow FCv1 brand: Rajdhani display type, DM Sans body copy, JetBrains Mono telemetry labels, and the fire/amber operational palette.
